Description

The Robot has to find the exit of the Maze by using proximity sensors and odometry, and follow the inner wall while avoiding obstacles. The method is based on pure PID control and on-board sensing. The projects goals were to implement the low-level sensor sockets in C++, I2C communincations, and the ROS architecture.
